# unit2px
convert unit (vh/vw) to px


## install

``` bash
npm i -g unit2px

# or

yarn add global unit2px
```


## use

```bash
$ unit2px [options]

Options:
  -d <dir...>         directory to be convert (default: ['.'])
  -t <type...>        file types to be convert (default: ['.html', '.css'])
  -w <width>          client width (default: 375)
  -h <height>         client height (default: 812)
```
